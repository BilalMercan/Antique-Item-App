
# Antique Items App

#### EN

Application that saves antiques, photo, owner and date in Sqlite database.

Features used in the app;

- View Binding
- Sqlite
- Snackbar
- Bitmap 

##### Layout Design
- RecyclerView
- Menu layout page is created in the app bar, items are added and activity transitions are provided.
